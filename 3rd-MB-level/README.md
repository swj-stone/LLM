## 结构特征

**模型结构 - Transformer Decoder（类GPT）：**
- 输入: "你好"
- ↓
-【Token Embedding】: 将token映射为384维向量
-【Position Embedding】: 添加位置信息
- ↓
-【Transformer层 × 4】: 自注意力机制 + 前馈网络
- ↓
-【LayerNorm + LM Head】: 输出词汇表概率分布
- ↓
- 输出:

** 训练过程 - 自回归语言建模**
- 目标：根据前文预测下一个token
- 损失函数：交叉熵（忽略pad位置）
- 优化器：AdamW + 余弦退火学习率调度
- 重复数据加载：30轮，每轮约250批次

**生成文本 - 自回归采样**
- 输入: "我"
- 第1步: 模型预测 → "是"
- 第2步: 用"我是"预测 → "三"
- 第3步: 用"我是三"预测 → "体"
- 第4步: 模型卡在重复循环 → "体体体体..."
- 问题：模型陷入重复生成，无法输出完整句子

## 改进方向

**扩大数据集**（最主要）：
- 采用公开真实数据集，而非自己生成
- 尝试至少MB级别的数据
- 数据增强、随机打乱句子顺序、同义词替换

**模型升级**：
- 增加beam search解码替代随机采样
- 引入top-p采样（nucleus sampling）

**训练方式**：
- 数据划分：train/val/test = 0.8/0.1/0.1
- 学习率调度：warmup + 余弦退火
- 增加训练轮数
- 梯度裁剪防止梯度爆炸
